## Usage

### Linux

```bash
sudo apt install python3 python3-pip
pip3 install -r requirements.txt
python3 main.py
```

### Windows

Download Python >= 3.7 from [here](https://www.python.org/downloads/)

```cmd
pip install -r requirements.txt
python main.py
```
